# EmulatorGPT

A game emulator using the OpenAI API (GPT-4o)

## Instructions

To run:

```bash
python main.py
```

## Sample list of games to try out

- Pokemon Red/Blue/Yellow

- Fallen London

- Rogue
- Dragon Quest (Dragon Warrior)
